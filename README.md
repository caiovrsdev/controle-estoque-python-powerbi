# Controle de Estoque com Python e Power BI

Este é um projeto simples de controle de estoque, onde:

- Usei **Python** para gerar e manipular dados fictícios de estoque.  
- Usei **Power BI** para criar um dashboard interativo que mostra o valor total em estoque, quantidade por categoria e detalhes dos produtos.

---

## Estrutura do projeto

- `dados/estoque.csv`: arquivo CSV com os dados fictícios do estoque.  
- `scripts/controle_estoque.py`: script Python para criar/manipular os dados.  
- Dashboard no Power BI baseado neste CSV para visualização dos dados.
